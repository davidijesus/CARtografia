# Glossário

| Termo | Definição |
| --- | --- |
| CAR | Cadastro Ambiental Rural, registro público eletrônico nacional obrigatório para imóveis rurais, previsto no Código Florestal. |
| SiCAR | Sistema Nacional de Cadastro Ambiental Rural, infraestrutura que apoia inscrição, consulta, análise e regularização do CAR. |
| Análise automatizada | Etapa em que regras e sistemas processam cadastros sem exigir análise manual detalhada para todos os casos. |
| Análise semiautomática | Etapa em que a automação não resolve sozinha e a analista precisa revisar evidências, camadas e inconsistências. |
| Suficiência cartográfica | Condição em que bases geoespaciais são atuais, confiáveis e completas o bastante para apoiar decisão técnica. |
| Índice de Suficiência Cartográfica | Avaliação estruturada da suficiência cartográfica de um caso, com status, justificativa e recomendação. |
| Dossiê CARtográfico | Artefato técnico revisável que organiza evidências quando a base é suficiente ou quase suficiente para decisão. |
| Kit de Atualização Cartográfica | Pacote técnico gerado quando a base é insuficiente, com diagnóstico, recorte, critérios e encaminhamento. |
| Canal de Acionamento Institucional | Fluxo estruturado para enviar dossiês e kits ao setor certo, com dono, SLA, status e histórico. |
| APP | Área de Preservação Permanente, protegida nos termos da legislação ambiental brasileira. |
| Reserva Legal | Área do imóvel rural que deve manter cobertura de vegetação nativa conforme regras do Código Florestal. |
| Uso e cobertura do solo | Classificação de como a superfície terrestre está ocupada ou coberta, como vegetação, agricultura, água ou área urbana. |
| Base de referência | Camada geoespacial usada como referência técnica para comparação, validação ou decisão. |
| Sensoriamento remoto | Obtenção de informações da superfície terrestre por sensores, como satélites, aeronaves ou drones. |
| GeoAI | Uso de inteligência artificial em problemas geoespaciais, como classificação, detecção de mudança e priorização. |
| Bem Público Digital | Software, dado, modelo, padrão ou conteúdo aberto, aderente a leis e boas práticas, com potencial de gerar valor público. |
| Interoperabilidade | Capacidade de sistemas trocarem dados e funcionarem juntos por padrões, APIs, metadados e formatos compatíveis. |
| Revisão humana | Validação técnica realizada por pessoa competente antes de decisão ou encaminhamento institucional. |
| Rastreabilidade | Capacidade de saber quais dados, fontes, versões, usuários e decisões participaram de um resultado. |
| SNIF | Sistema Nacional de Informações Florestais, fonte pública de dados e informações sobre florestas no Brasil. |
| MapBiomas | Iniciativa multi-institucional que produz séries de mapas de uso e cobertura da terra. |
| INPE/Brazil Data Cube | Projeto do INPE para produzir cubos de dados de observação da Terra e apoiar análise por séries temporais. |
| QA | Quality assurance, processo de verificação de qualidade técnica de entregas, bases e artefatos. |
| SLA | Service level agreement, prazo ou nível de serviço esperado para resposta e tratamento de uma demanda. |
| Webhook | Mecanismo de integração em que um sistema envia evento automaticamente para outro. |
| Metadados | Informações sobre um dado, como fonte, data, método, escala, resolução e limitações. |
